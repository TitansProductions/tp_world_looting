# I cant see any images, why?
------------------------------------------------------------------------------------------

Go to the tp-worldlooting/html/js/script.js -> line 13

You will need to change path to your inventory name.

As default, the path is `nui://tp-base/html/img/items/`, if you are using inventory hud,
the path would be something like this: `nui:esx_inventoryhud/html/img/items/`
